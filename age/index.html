Age Suitability

Kings is a scorekeeper for a family card game. It helps players track scores. It does not play the game for you.

Content Summary

Gambling No betting, wagering, prizes, or gambling features.
Chat No chat, direct messages, or social feed.
Public content No public user posts or public profile pages.
Ads No ads.
Web browsing No open web browser inside the app.
Accounts No separate Kings account. Optional sharing uses Apple's iCloud flow.

For Parents and Reviewers

Kings tracks scores for a card game played in person. The app stores player names, game scores, game history, and player stats so the table can keep score.

The app does not include game betting, money, prizes, chat, or public posts.
