Kings support for game night.

Kings is a focused iPhone and iPad scorekeeper for the Kings card game. This site helps players start games, score hands, understand family scorebook sharing, and find privacy information.

Get Support Read Privacy Policy

Start Scoring

Create a game, choose 2 to 6 players, score each hand, and keep the table moving without paper math.

Learn the Rules

New to Kings? Read the plain-language guide for the goal, deal, wild cards, turns, and scoring.

Check Age Info

Kings has no betting, chat, public posts, ads, or open web browser. It is a scorekeeper for in-person play.

Review History

Completed games stay in history so you can review final standings, hand results, and player stats.

Share Carefully

Optional iCloud scorebook sharing uses Apple's iCloud and CloudKit systems. Player photos are intended to stay local.

Need help?

Use the support page for common questions and troubleshooting. If you still need help, you can open a public support request on GitHub. Do not include private personal information in public support requests.
